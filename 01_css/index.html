Playing with CSS

  • The intial release of CSS was December 17, 1996.
  • The CSS em unit was originally derived from the width of the capital letter M.
  • Eric Meyer has a great CSS reset snippet. This block of code helps reduce browser inconsistencies, making it easier to style across all browsers.
  • DRY is an acronym in development that stands for Don't Repeat Yourself. Many languages, including CSS, suggest keeping your code DRY.
  • If you want to include a font, you'll likely need to use @font-face.
